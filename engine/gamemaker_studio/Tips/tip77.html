Events

If you have multiple objects with the same block of code in the same event, you should consider using a Parent object. Creating a parent with this event and code, means that its children will "inherit" the event, making modifying code easier as a change to the parent code block will affect all the "child" objects too. Note that if the "child" object needs specific code in the same event, you should call event_inherited() first to call the parent code-block.
