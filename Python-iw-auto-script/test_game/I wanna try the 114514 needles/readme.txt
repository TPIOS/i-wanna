2.操作方法
　Shift　　　  　　　　　　　：ジャンプ(２段ジャンプ可)
　←→キー　   　　　　　　　：左右移動
　Zキー　　　  　　　　　　　：攻撃(セーブ)
　Rキー　      　　　　　　　：リトライ
　Qキー　　　　　　　　　　　：自殺
　F2 (PCによってはctrl+F2)　 ：タイトル画面に戻る
　Esc　　　　　　　　　　　　：終了
　Alt+Ins      　　　　　　　：終了
　F9 (PCによってはctrl+F9)   ：スクリーンショット


3.ゲーム説明
　3-1.概要
　　　タイトルの通り114514個の上下針を越えたらクリアです
　　　セーブは自動です。
　　　
　3-2.難易度
　　　色々な意見があると思いますが、上下針さえ越えられれば誰でもクリア可能なため初心者向けだと思っています
